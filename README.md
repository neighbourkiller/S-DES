# 实现了S-DES算法
## 功能
1.加密：输入8bit明文和10bit密钥

2.解密：输入8bit密文和10bit密钥

3.暴力破解：输入相同长度的明文密文，获得多个可能密钥

4.明文输入可以是二进制数，可以是单个ASCII码表中的字符
